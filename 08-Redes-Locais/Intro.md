# Redes Locais

## Padrão IEEE802

O padrão IEEE 802 foi criado em 1980 pelo IEEE com o objetivo de padronizar redes locais e metropolitanas, definindo as camadas Física e de Enlace do modelo Modelo OSI. Isso permitiu que diferentes fabricantes produzissem equipamentos compatíveis entre si.

Foram definidos grupos de trabalho para especificar cada tecnologia de acesso ao meio.
Iniciou-se em fevereiro de 1980, com:
* **802.1** - Higher Leyer Interface
* **802.2** - Logical Link Control
* **802.3** - CSMA/CD - similar ao Ethernet

Após um ano, foram adicionados:
* **802.4** - Token Bus (barramento com token)
* **802.5** - Token Ring (anel com token)

Ao longo do tempo, vários grupos de trabalho foram criados. Algumas tecnologias caíram em
desuso, e os respectivos grupos de trabalho foram desativados.
Os grupos de trabalho ativos atualmente são (`http://www.ieee802.org`):

- **802.1** Higher Layer LAN Protocols Working Group
- **802.3** Ethernet Working Group (Ethernet)
- **802.11** Wireless LAN Working Group (Wi-fi)
- **802.15** Wireless Personal Area Network (WPAN) Working Group (Bluetooth)
- **802.18** Radio Regulatory TAG
- **802.19** Wireless Coexistence Working Group
- **802.24** Vertical Applications TAG

Cada um desses grupos de trabalho IEEE 802 criou várias especificações, padronizando os
serviços de cada função.
