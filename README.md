# REDES DE COMPUTADORES
* ***CURSO:*** TADS IFPR-Foz 
* ***PROFESSOR:*** Julio Cesar Royer
* ***SEMESTRE:*** 3º
* ***HORA AULA:*** 80h
* ***HORA RELOGIO:*** 67h

### Objetivo:
Capacitar o aluno para o entendimento das tecnologias utilizadas para a
interligação em redes de computadores

### Ementa:
- Introdução às redes de computadores e a Internet
- Arquiteturas de redes decomputadores: modelos OSI e TCP/IP.
- Protocolo de Aplicação HTTP
- Serviço DNS
- Protocolos de Transporte TCP e UDP
- Protocolo IP, Endereçamento IP, Roteamento IP
- Protocolos e serviços ICMP, DHCP, NAT e ARP.
- Redes locais cabeadas e sem fio.
- Equipamentos de rede e meios físicos de comunicação.
- Laboratórios e práticas com comandos e ferramentas de rede em ambientes Linux
